# Procom 蔵書検索

## 概要
Procom 蔵書検索は、プログラマーズコミュニティ部の蔵書データを検索・閲覧できるWebアプリケーションです。キーワードや詳細条件で本を検索し、一覧や詳細を表示できます。

## ファイル構成

- index.html …… トップページ・検索画面
- books.html …… 蔵書一覧ページ
- result.html …… 検索結果表示ページ
- img/NoImage.png …… 書影がない場合の画像

## 主な機能

- キーワード・詳細条件による蔵書検索
- 検索結果・蔵書一覧の表示

## 使い方

1. `index.html` をブラウザで開きます。
2. 「簡易検索」または「詳細検索」ボタンで検索フォームを切り替え、キーワードや条件を入力して「検索」ボタンを押します。
3. 検索結果が `result.html` に表示されます。
4. 「本一覧」ページ（books.html）では全蔵書の一覧が見られます。

### 主な画面
- **トップページ（index.html）**: 検索フォーム、検索条件の入力
- **蔵書一覧（books.html）**: すべての蔵書を一覧表示
- **検索結果（result.html）**: 条件に合致した本のリストを表示
